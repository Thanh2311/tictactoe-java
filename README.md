# Simple TicTacToe Game #
by Tim Tsu

## Getting started ##
### Prerequisites ###
* JRE 1.6
* Maven 2 or Gradle

### Build and Run ###
#### Maven ####
    mvn package
    java -jar target/tictactoe-1.0.jar
#### Gradle ####
    gradle installApp
    ./build/install/tictactoe-java/bin/tictactoe-java
